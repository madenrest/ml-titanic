> **Цель проекта:** построить модель для предсказания выживаемости пассажиров «Титаника» и интерпретировать ключевые факторы, влияющие на исход.

## О проекте

классический пайплайн
- EDA: анализ распределений, поиск зависимостей, визуализация 
- Предобработка: заполнение пропусков, кодирование категориальных признаков, создание новых фич 
- Моделирование: логистическая регрессия с интерпретацией коэффициентов
- Оценка: accuracy, precision, recall

### Установка и запуск

git clone https://github.com/madenrest/ml-titanic.git
cd ml-titanic
установить виртуальное окружение
pip install -r requirements.txt

### Ключевые результаты
Accuracy: 0.83
Precision: 0.83 - из всех предсказанных выживших столько выжили на самом деле
Recall: 0.74 - модель назодит ~74%  выживших
Главные факторы выживаемости: пол, класс обслуживания
